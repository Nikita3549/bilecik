Бот які правярае кошты авіябілетаў Белавіа і паведамляе пра зніжкі

### Крыніца дадзеных і абмежаванні API

Бот бярэ цэны праз GraphQL-аперацыю FlightsMinPricesInPeriodRT (эндпоінт webapi.belavia.by/graphql/query/nemo).

Абмежаванне daysCount (пагінацыя). Параметр daysCount жорстка прывязаны да значэння 7 — гэта памер акна пагінацыі на сайце Белавіі. Толькі пры daysCount: 7 API вяртае карэктныя цэны: пры 8 і больш яны скажаюцца, пры 6 усе становяцца null. Таму ў кодзе гэта канстанта, а дыяпазон дат карыстальніка рэжацца на вокны па 7 дзён
